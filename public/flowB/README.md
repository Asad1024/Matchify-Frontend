## Image guidelines

- **personality_turn_on / energy_vibe:** Expressive close-up or chest-up studio portraits.
- **physical_attraction:** Upper-body portraits.
- **face_shape_preference:** Close-up face shots.
- **eye_shape_preference:** Eye-focused portraits.
- **lip_shape_preference:** Lower-face crop.
- Crop 1:1 or 3:4, face centered. Export 1080×1080 JPEG, &lt;300KB.
